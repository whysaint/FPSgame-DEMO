Quick help: how to use this asset?
----------------------------------

- Select one of the provided materials in SkyboxPlus/Resources/SkyboxPlus/Materials and assign it to the Skybox material in Lighting Settings in Unity menu.
- Run Demo scene included for a quick demonstration of the different materials.
- Read the documentation folder for a description of each parameter of the material and customize it to your needs.

Universal Rendering Pipeline (URP)
----------------------------------

Skybox Plus is compatible with built-in and URP pipelines, however the the demo scene included with the asset was created for built-in pipeline.
If you open the demo scene with URP, the terrain won't show correctly (pink) but that's expected. The Skybox Plus materials will work fine in your scene.
